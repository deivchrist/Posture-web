Semana 10: Fabricación Digital

De la preparación a la fabricación

En esta fase, aplicamos los conceptos de clasificación y preparación de piezas para la fabricación digital. Analizamos el modelo 3D de nuestro proyecto "Posture" definiremos los procesos, materiales y roles para la construcción del prototipo físico.

Actividad 1: Clasificación de Piezas del Proyecto

Analizamos nuestro modelo digital completo y decidimos el método de fabricación óptimo para cada componente, justificando la elección del material y el proceso.

Pieza Tipo de fabricación Justificación Material sugerido
Collar / Banda Ergonómica Impresión 3D La pieza tiene una geometría 3D orgánica y curva, diseñada para adaptarse al cuello. Es imposible de fabricar con corte láser. Requiere ser un solo cuerpo sólido y resistente. PLA (por su rigidez)
Soporte de Electrónica Impresión 3D Contiene geometrías 3D complejas: curvatura para la nuca, cavidades internas, postes para tornillos y encajes precisos para el PCB y la batería. PLA (por su rigidez)
Caja de Componentes Corte Láser Aunque el modelo 3D muestra una caja, decidimos fabricarla con cartón. El diseño vectorial 2D permite un plegado rápido (usando *living hinges*) para formar la carcasa. Es más rápido y económico para esta etapa. Cartón Piedra / MDF 3mm

Actividad 2: Ajustes Previos a la Fabricación

Revisamos y preparamos los modelos digitales, considerando tolerancias, optimización de material y orientación de impresión.

Ajustes de Impresión 3D (PLA / PETG)

Para las piezas 3D, la orientación es clave. El Collar (175.24mm x 182.62mm) se imprimirá "acostado" para maximizar la adhesión entre capas y la resistencia a la flexión.

El Soporte de Electrónica (129.54mm x 58.77mm) se imprimirá con su base plana tocando la cama de impresión para eliminar la necesidad de soportes internos, optimizando el tiempo y el material.

Ajustes de Corte Láser (Cartón / MDF)

Para la Caja de Componentes, preparamos un layout vectorial 2D. Las piezas se organizaron para optimizar el material en una plancha de 100.0mm.

Se consideró una tolerancia de "quemado" (kerf) de 0.10mm, como se indica en el archivo, para asegurar que las pestañas y ranuras encajen perfectamente tras el corte.

Actividad 3: Asignación de Roles del Equipo

Para optimizar el flujo de trabajo de fabricación, definimos las siguientes responsabilidades para los miembros del equipo.

Rol / Responsabilidad Integrante Asignado
Responsable de Impresión 3D: Preparar archivos STL, laminar (slicing) y gestionar tiempos de impresión. Deivid Christian
Responsable de Corte Láser: Generar archivos SVG/DXF, optimizar el layout y operar la cortadora. Bonifacio Zevillano
Responsable de Documentación: Registrar todo el proceso, tomar fotografías y actualizar el sitio web. Deivid Christian
Responsable de Ensamblaje: Post-procesar piezas, probar encajes y ensamblar la electrónica. Bonifacio Zevillano
